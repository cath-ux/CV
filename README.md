## Quick Start

### View Locally
1. Open `Cathresumehtml.html` in any web browser
2. No installation or server required

### View Online (Deployment Options)

**Option 1: GitHub Pages (Recommended)**
1. Create a GitHub repository named `Portfolio` (your repo: `cath-ux/Portfolio`)
2. Rename `Cathresumehtml.html` to `index.html`
3. Upload `index.html` to the repository root
4. Push to GitHub
5. Enable GitHub Pages in repository settings (Settings → Pages → Source: main branch)
6. Your site will be live at: `https://cath-ux.github.io/Portfolio/`

**Note:** Only the `index.html` file needs to be published. The README.md is optional for GitHub.

**Option 2: Netlify**
- Drag and drop the HTML file at netlify.com
- Get a free hosted URL instantly

**Option 3: Vercel**
- Upload to Vercel for automatic deployments

**Option 4: Web Server**
- Upload to any web hosting provider (Bluehost, GoDaddy, etc.)

## Customization

### Update Contact Information
Edit the contact section in the HTML file:
```html
<!-- Find and replace contact details -->
```

### Change Colors
Edit the CSS variables at the top of the `<style>` section:
```css
:root {
    --navy: #0D1B2A;
    --gold: #C9A84C;
    --cream: #FAF7F2;
    /* etc... */
}
```

### Add/Remove Sections
Edit the HTML content between the `<body>` tags to add or remove sections.
